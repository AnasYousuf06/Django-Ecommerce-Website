 My Django Ecommerce Project

A simple Django web application for managing recipes and food orders.

## Features
- User authentication
- Product management
- Shopping cart
- Order management

## Installation
1. Clone the repo
2. Create virtual environment
3. Install dependencies
4. Run migrations
5. Start the server

## Usage
Visit http://127.0.0.1:8000/ and enjoy the project!
